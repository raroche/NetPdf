# API reference

The public NetPdf API surface, generated from the shipping assemblies' XML documentation.

Start here:

- [`HtmlPdf`](NetPdf.HtmlPdf.yml) — the static conversion facade (`Convert` / `ConvertAsync`).
- [`HtmlPdfOptions`](NetPdf.HtmlPdfOptions.yml) — page geometry, base URI, metadata, feature flags.
- [`HtmlPdfException`](NetPdf.HtmlPdfException.yml) — the typed render error carrying a diagnostic `Code`.
- [`HyphenationRegistry`](NetPdf.Hyphenation.HyphenationRegistry.yml) — the language-hyphenation extension seam
  that the `NetPdf.Languages.*` packs register with.

Use the table of contents on the left to browse the full namespace tree.
